# module
##项目内容
这是一个开源的springboot开源框架
- 包含logbak日志
- druid数据库连接
- redis工具
- 异常统一处理接口
- swagger
- mybatis-plus
- mybatis-plus-generator 模板
- 重试切口
- 令牌桶切口
- xss过滤且

##项目结构
- module 
    - frame 项目配置
    - mod  业务代码

##使用说明
修改包名要修改
- MybatisPlusConfig
- SwaggerConfig
- resources/templates
- 配置文件 MybatisPlus的扫描
